# Knock CRM TM_팀장 분석 문서

## 1. 파일 개요

### 기본 정보
- **파일명**: Knock_CRM_mng_v.1.62_5_TM_팀장.xlsm
- **버전**: v.1.62
- **목적**: 치과 CRM 시스템의 TM(Tele-Marketing) 팀장용 관리 도구
- **대상 사용자**: TM 팀장급 관리자
- **주요 기능**: DB 배포 관리, INPUT DB 입력/관리, 통계 조회, DentWeb 연동, 예약/내원 정보 관리

### 시스템 특징
- Excel VBA 기반 데이터베이스 관리 시스템
- ODBC 연결을 통한 Oracle 데이터베이스 접근
- DentWeb(치과 관리 시스템)과의 양방향 연동
- 실시간 DB 배포 및 회수 관리
- 통계 및 성과 분석 기능

## 11. 비즈니스 프로세스 요약

```
[DB 입력] → [DB 배포] → [TM 통화] → [예약] → [내원] → [통계]
    ↓           ↓          ↓         ↓        ↓       ↓
INPUT_DB → DB_배포 → TM_CALL_LOG → DentWeb → 내원 → 대시보드
```

### 일일 업무 흐름

1. **오전 9시**: DentWeb 연동 (전일 예약 데이터 동기화)
2. **오전 10시**: 신규 DB 입력 (InputDB_작업 → InputDB_입력 → DB Upload)
3. **오전 11시**: DB 배포 Query → TM 배정 → DB Upload
4. **오후 1시~5시**: TM 통화 작업 (별도 TM용 파일)
5. **오후 6시**: 당일 통계 확인 (대시보드)
6. **월말**: DB 결산, DB 업체 결제

## 12. 결론

이 시스템은 **치과 CRM의 핵심 운영 도구**로서, TM 팀장이 일일 DB 배포, 성과 관리, 품질 관리를 효율적으로 수행할 수 있도록 지원합니다. 특히 DentWeb과의 연동을 통해 예약부터 내원까지의 전체 고객 여정을 추적하고, 이를 바탕으로 DB 품질 및 TM 성과를 정량적으로 평가할 수 있습니다.
